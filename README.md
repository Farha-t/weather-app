# Weather App 🌤️

A Python-based weather application using OpenWeatherMap API.

## Features
- Search weather by city name
- Displays temperature, feels like, humidity, wind speed and condition
- Error handling for invalid city names

## Technologies Used
- Python
- Requests library
- OpenWeatherMap API

## How to Run
1. Install requests: pip install requests
2. Run: python weather.py
3. Enter city name when prompted
